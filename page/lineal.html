Estructuras de Control

If

La estructura If se utiliza para tomar decisiones en función de una condición, Permite ejecutar un bloque de código solo si la condición especificada es verdadera. Si la condición es falsa, el bloque de código dentro del if no se ejecuta. Esta estructura es fundamental para controlar el flujo de un programa y permite que este responda de manera diferente según las circunstancias.

Else

La estructura else se utiliza para ejecutar un bloque de código cuando la condición de un if es falsa. Permite manejar situaciones alternativas y tomar decisiones en el flujo del programa, asegurando que se ejecute un código específico si la condición planteada no se cumple.

While

La estructura while se utiliza para ejecutar un bloque de código de manera repetitiva mientras una condición especificada sea verdadera. Es útil para situaciones en las que no se sabe de antemano cuántas veces se debe repetir el bloque de código, ya que la ejecución continuará hasta que la condición se evalúe como falsa. Esto permite crear bucles que pueden iterar hasta que se cumpla una determinada condición.

Case

>

La estructura case se utiliza para seleccionar y ejecutar uno de varios bloques de código basándose en el valor de una expresión. Es especialmente útil cuando se tienen múltiples condiciones que se pueden evaluar de manera más clara y organizada que con múltiples if y else if. Permite manejar diferentes casos de forma eficiente, facilitando la lectura y mantenimiento del código.

Do While

La estructura do while se utiliza para ejecutar un bloque de código al menos una vez y luego repetir la ejecución mientras una condición especificada sea verdadera. A diferencia de while, que evalúa la condición antes de ejecutar el bloque de código, do while garantiza que el código se ejecute al menos una vez antes de verificar la condición. Esto es útil en situaciones donde se necesita que el bloque de código se ejecute antes de realizar la evaluación de la condición.
